# Reference Designs

Drop premium funnel screenshots here to use as reference when building new pages with `/new-funnel`.

## How to use

1. Save screenshots of high-quality funnel pages as `.png` or `.jpg`
2. Name them descriptively, e.g. `hero-section-dark.png`, `pricing-card-clean.png`
3. Run `/new-funnel` — Claude will visually read everything in this folder

## Tips

- Full-page screenshots work best
- You can have multiple references (e.g. one for layout, one for color style)
- Organize by type if needed: `hero/`, `pricing/`, `testimonials/`
