# Kristina / Splash Swim School of Auburn: private swim lessons in Auburn and Opelika, AL

## What this is
Kristina ("Miss Kris") runs Splash Swim School of Auburn, offering private swim lessons for infants, toddlers, and young children in Auburn and Opelika, Alabama. This folder holds the main marketing site plus a "Bath Time Without Tears" lead magnet landing for The Miss Kris Method.

## Architecture
- `kristina-v2.html` : main marketing / sales page (lives at splashauburn.com), floating glassmorphism nav, hero video, Playfair Display + Montserrat
- `leadmagnet.html` : lead-magnet funnel for "Bath Time Without Tears: The Miss Kris Method", uses the full GHL-breakout shell so it can be embedded edge-to-edge
- Tech: HTML + embedded CSS/JS, GitHub Pages, GHL iframe embed (standard pattern across all client funnels in this repo)

## Conventions
- Brand colors: blue `#2a5a84` primary, deep blue `#1e4466`, sky `#8ec6d8` secondary, warm bg `#f6f5f1`
- Fonts: Playfair Display (headings) + Montserrat (body)
- Floating capsule nav (`border-radius: 50px`) with glass background and tiered shadows
- The `v2` suffix on the main file is intentional, treat it as the canonical main page
