# Ignacio / CS Infraestructura: industrial construction in Chile (Spanish-language)

## What this is
CS Infraestructura SpA is a 24-year-old Chilean industrial construction firm, positioned as the #1 builder of industrial refrigeration plants ("frigoríficos industriales") plus treatment plants and institutional construction. This folder holds the corporate marketing site plus a separate product landing for "Ecosipboard".

## Architecture
- `cs.html` : main corporate site (Spanish, `lang="es"`) for CS Infraestructura SpA
- `ecosipboard.html` : product landing for "Ecosipboard" ("un panel que hace el trabajo de tres"), Spanish, separate green palette, includes Google Ads gtag `AW-17979536691`
- Tech: HTML + embedded CSS/JS, GitHub Pages, GHL iframe embed (standard pattern across all client funnels in this repo)

## Conventions
- `cs.html` brand: red `#d32f2f` primary on white/gray scale, Instrument Serif + Inter, 1200px max container, 68px nav logo
- `ecosipboard.html` brand: green palette (`#1a3d2b`, `#2d5a3d`, `#4a9c6d`, mint `#7ac9a0`) with Inter + Space Grotesk fonts, separate shadow + green-glow system
- Both pages are Spanish, copy uses formal/B2B tone
