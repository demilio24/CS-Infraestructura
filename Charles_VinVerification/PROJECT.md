# Charles / Miami VIN Verification: same-day Florida VIN inspections

## What this is
Same client as `Charles_Notary` (veteran-owned), positioned as a separate service line: licensed VIN verification for cars, motorcycles, boats, trailers, and rebuilt titles across Miami-Dade County. Single-page sales site.

## Architecture
- `charles-vin.html` : standalone marketing / sales page for the VIN verification service
- Tech: HTML + embedded CSS/JS, GitHub Pages, GHL iframe embed (standard pattern across all client funnels in this repo)

## Conventions
- Brand colors: blue `#1E5FA8` primary (lighter / brighter than the notary site's navy), orange `#F97316` accent for CTAs and labels
- Fonts: Instrument Sans (headings) + DM Sans (body)
- Radius tokens 6/12/20/32/full, dedicated card and accent shadows
- 72px fixed nav height
