# Blake / Intuitive Swimming: life-saving swim lessons in Metro Denver

## What this is
Blake runs Intuitive Swimming, teaching water safety to kids ages 6 months and up across Englewood, Littleton, and South Denver (operating since 2021). This folder holds the marketing site plus a linktree-style booking landing.

## Architecture
- `blake.html` : main marketing / sales page (Metro Denver service area, age and method positioning)
- `blake-linktree.html` : standalone "book with Blake" landing styled as a single CTA card on a blue gradient, intended for bio links
- Tech: HTML + embedded CSS/JS, GitHub Pages, GHL iframe embed (standard pattern across all client funnels in this repo)

## Conventions
- Brand colors: navy/blue scale (`--blue #0055aa`, `--blue-dark #003d7a`, light tints), red accent for safety messaging
- Fonts: DM Serif Display (headings) + DM Sans (body)
- Container max 1080px, section padding 88px, three shadow tiers with blue glow
- Linktree uses a separate brighter palette (`#0077B6`, `#00B4D8`) on a full-bleed gradient body
